Learn about this element

When a flow runs, record changes are executed and saved to the database as part of a transaction. This element rolls back the current transaction and cancels all its pending record changes. To roll back changes based on business logic, add this element in a Decision element’s outcome. To roll back changes when an error occurs, add this element in a Fault path. To roll back all record changes in a transaction, add this element to the end of the transaction.
